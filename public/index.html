Screen Racers

Level: 0

Screen Racers is a game that is consists of challenges to help you learn the basic commands of using a screen reader. Click on a challenge to begin learning!


Screen racers was designed and developed by Serena Buxton for the Super Human Computing Lab under the advisment of Shaun Kane. The purpose of the application is to explore ways to lighten the cognitive load of a sighted person learning to use a screen reader. The hope is that making the process more engaging will encourage more software developers to learn the basics so that they can develop more accessible websites.
